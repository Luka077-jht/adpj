# ✈️ 爱的迫降角色评分系统

## 🎬 项目介绍
这是一个专门为韩剧《爱的迫降》设计的角色评分系统，采用虎扑风格的界面设计。

## ✨ 主要功能
- **角色评分**: 为剧中主要角色进行五星评分
- **实时统计**: 显示评分数据和排行榜
- **角色热梗**: 展示角色相关的热门梗和评论
- **筛选搜索**: 支持按角色类型和评分范围筛选

## 🚀 快速开始

### 安装依赖
```bash
pip install -r adpj_requirements.txt
```

### 运行应用
```bash
streamlit run 爱的迫降.py
```

### 访问应用
打开浏览器访问: http://localhost:8501

## 📊 角色列表
- 尹世理 (女主角) - 韩国财阀继承人
- 李正赫 (男主角) - 朝鲜军官
- 徐丹 (女配角) - 李正赫的未婚妻
- 具承俊 (男配角) - 尹世理的前男友
- 表治秀 (男配角) - 李正赫的部下
- 金舟墨 (男配角) - 李正赫的部下

## 🛠️ 技术栈
- **前端框架**: Streamlit
- **数据处理**: Pandas, NumPy
- **界面风格**: 虎扑风格CSS
- **部署方式**: Streamlit Cloud / 本地部署

## 🎯 使用说明
1. 运行应用后，在左侧导航栏选择"角色评分"
2. 查看角色信息，包括图片、描述和评分
3. 使用五星评分系统为角色打分
4. 查看实时统计和排行榜
5. 使用筛选功能查找特定角色
